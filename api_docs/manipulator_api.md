# Класс `Manipulator`

Доступ через `robot.manipulator` (в случае `FourWDCar`) или напрямую при инициализации (если робот имеет только манипулятор).

## Инициализация

Обычно манипулятор является частью робота:

```python
from fourWDCar import FourWDCar

robot = FourWDCar()
manip = robot.manipulator
```

## Проверка режима API

Все методы манипулятора автоматически проверяют, включён ли режим API у робота. Если нет – выбрасывается исключение:

> **"API mode is disabled. Switch Input Mode to API."**

## Методы управления захватом (греббером)

| Метод | Описание |
|-------|----------|
| `lift() -> None` | Поднимает захват вверх (относительно текущего положения стрелы). Движение происходит с фиксированной скоростью (без временного параметра). |
| `lower() -> None` | Опускает захват вниз. |
| `set_angle(angle: float = 45.0) -> None` | Устанавливает угол раскрытия захвата (между пальцами) в градусах.<br>`angle` – угол раствора (0 = полностью закрыт, максимальное значение – около 70–90°, зависит от модели). |
| `close(angle: float = 45.0) -> None` | Закрывает захват до указанного угла. Фактически синоним `set_angle`, но логически означает «зажать» объект. |
| `open_full(time: float = 2.0) -> None` | Полностью раскрывает захват (до максимального угла).<br>`time` – время движения в секундах (по умолчанию 2 секунды).<br>**Блокирующий** вызов – скрипт ожидает завершения движения. |
| `close_full(time: float = 2.0) -> None` | Полностью закрывает захват (до минимального угла, обычно 0).<br>`time` – время движения.<br>**Блокирующий** вызов. |

```python
manip.lift()  # поднять захват
manip.lower()  # опустить захват

manip.set_angle(30)   # раскрыть на 30 градусов
manip.set_angle(0)    # полностью закрыть

manip.close(20)  # закрыть до 20 градусов (схватить объект)

manip.open_full(1.5)  # полностью раскрыть за 1.5 сек
manip.close_full(1.2)  # закрыть захват за 1.2 сек
```

## Методы управления стрелой (подъём/опускание руки)

| Метод | Описание |
|-------|----------|
| `arm_up(time: float = 2.0) -> None` | Поднимает стрелу манипулятора вверх (до верхнего предела).<br>`time` – время движения (секунды).<br>**Блокирующий** вызов. |
| `arm_down(time: float = 2.0) -> None` | Опускает стрелу вниз (до нижнего предела). |
| `move_arm(angle: float, time: float = 2.0) -> None` | Перемещает стрелу в указанное абсолютное положение (угол в градусах относительно нейтрали).<br>`angle` – целевой угол (обычно 0 – горизонталь, положительный – вверх, отрицательный – вниз, зависит от настройки).<br>`time` – время движения.<br>**Блокирующий** вызов. |
| `reset() -> None` | Сбрасывает манипулятор в начальное (дефолтное) положение: стрела в нейтраль, захват закрыт или открыт на определённый стартовый угол.<br>**Не блокирует** выполнение – движение происходит параллельно. |

```python
manip.arm_up(2.0)    # поднять руку за 2 сек
manip.arm_down(1.5)  # опустить за 1.5 сек

# перевести стрелу в положение 30 градусов вверх за 2 секунды
manip.move_arm(30, 2.0)

# затем опустить в -20 градусов (вниз) за 1 секунду
manip.move_arm(-20, 1.0)

manip.reset()  # сброс в исходное положение
```

## Полный список методов

| Метод | Описание | Параметры | Блокирующий |
|-------|----------|-----------|-------------|
| `lift()` | Поднять захват | – | Нет |
| `lower()` | Опустить захват | – | Нет |
| `set_angle(angle)` | Раскрыть захват на заданный угол (градусы) | `angle` (по умолч. 45) | Нет |
| `close(angle)` | Закрыть захват до угла (алгоритмически то же, что `set_angle`) | `angle` (по умолч. 45) | Нет |
| `open_full(time)` | Полностью раскрыть захват | `time` (сек, по умолч. 2.0) | Да |
| `close_full(time)` | Полностью закрыть захват | `time` | Да |
| `arm_up(time)` | Поднять стрелу вверх | `time` | Да |
| `arm_down(time)` | Опустить стрелу вниз | `time` | Да |
| `move_arm(angle, time)` | Переместить стрелу в указанный угол | `angle`, `time` | Да |
| `reset()` | Сбросить в начальное положение | – | Нет |

> **Блокирующие методы** останавливают выполнение Python-скрипта до окончания движения (используют `time.sleep` внутри адаптера). Это удобно для последовательных команд, но может замедлить сложные алгоритмы. Для параллельного выполнения используйте потоки или асинхронность.

## Примеры использования

### Пример 1. Захват и подъём объекта

```python
from fourWDCar import FourWDCar
import time

robot = FourWDCar()
manip = robot.manipulator

# Подъехать к объекту (команды движения платформы)
robot.platform.moveForward(30, ms=2000)

# Открыть захват, опустить стрелу, закрыть, поднять
manip.open_full(1.0)      # полностью раскрыть за 1 сек
manip.arm_down(1.5)       # опустить стрелу
manip.close_full(1.2)     # захватить объект
manip.arm_up(2.0)         # поднять объект

# Увезти объект
robot.platform.moveBackward(30, ms=2000)
robot.platform.stop()
```

### Пример 2. Точное позиционирование захвата

```python
# Раскрыть захват на 60 градусов
manip.set_angle(60)

# Закрыть до 10 градусов (удерживать объект)
manip.close(10)

# Поднять стрелу на 45 градусов вверх
manip.move_arm(45, 1.5)

# Вернуть всё в исходное состояние
manip.reset()
```

### Пример 3. Циклическое тестирование

```python
for i in range(3):
    manip.open_full(1.0)
    manip.close_full(1.0)
    manip.arm_up(1.0)
    manip.arm_down(1.0)
    print(f"Цикл {i+1} завершён")
```

## Особенности реализации

| Пункт | Описание |
|-------|----------|
| **Проверка API** | Все методы (кроме `reset`) вызывают `_check()`, который выбрасывает исключение, если манипулятор находится в режиме ручного управления (`InputControlMode.Keyboard` или другой). |
| **Ограничение скорости** | Время движения (`time`) ограничено диапазоном `[MinRotationSpeed, MaxRotationSpeed]` (обычно от 0.1 до 2 секунд). Если указать время меньше минимума, оно будет поднято до минимального; если больше максимума – ограничено максимальным. |
| **Плавность движения** | Движения по времени интерполируются – угол изменяется линейно от текущего к целевому за заданное время. |
| **Блокировка** | Пока выполняется любой блокирующий метод (с параметром `time`), другие команды манипулятора не могут быть запущены – последовательность гарантирована. |
| **Сброс** | `reset()` немедленно устанавливает целевые углы в стартовые значения (без временной интерполяции) и останавливает любые текущие корутины движения. |
